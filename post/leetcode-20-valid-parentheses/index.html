LeetCode 20 Valid Parentheses

By xdzhcs

2016年09月18日

题目

https://leetcode.com/problems/valid-parentheses/ >Given a string containing just the characters ‘(’, ‘)’, ‘{’, ‘}’, ‘[’ and ‘]’, determine if the input string is valid. The brackets must close in the correct order, “()” and “()[]{}” are all valid but “(]” and “([)]” are not.

题意

括号匹配问题。给出一个字符串，里面只有’(‘, ‘)’, ‘{’, ‘}’, ‘[‘6种字符，求括号是否合法匹配。

思路

用栈结构。

代码

public class Solution {
    public boolean isValid(String s) {
        Stack<Character> stack=new Stack<>();
        for(int i=0;i<s.length();i++){
            if(!stack.isEmpty()){
                switch ((char)stack.peek()) {
                case '(':
                    if(s.charAt(i)==')')stack.pop();
                    else stack.push(s.charAt(i));
                    break;
                case '[':
                    if(s.charAt(i)==']')stack.pop();
                    else stack.push(s.charAt(i));
                    break;
                case '{':
                    if(s.charAt(i)=='}')stack.pop();
                    else stack.push(s.charAt(i));
                    break;
                default:
                    break;
                }
            }else {
                stack.push(s.charAt(i));
            }
        }
        return stack.isEmpty();
    }
}
